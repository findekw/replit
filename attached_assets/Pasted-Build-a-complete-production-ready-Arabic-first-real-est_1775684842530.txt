Build a complete production-ready Arabic-first real estate marketplace web application inspired by finde.co, but more modern, more scalable, and designed as a SaaS platform for real estate offices in Kuwait and the Gulf.

IMPORTANT:
- The website must be RTL-first and Arabic-first.
- The design must be premium, modern, clean, fast, and conversion-focused.
- Use the FINDE branding style and use the logo/assets from finde.co as visual inspiration.
- The homepage search experience should feel inspired by finde.co, but the UI/UX must be better, cleaner, and more modern.
- Build it as a real multi-role platform, not just a static property listing site.

GOAL:
Create a real estate marketplace where:
1) end users search for properties,
2) real estate offices/brokers can subscribe, manage listings, receive leads, and get their own office page,
3) admins can manage the full platform.

TECH STACK:
- Frontend: Next.js latest with App Router + TypeScript
- Styling: Tailwind CSS + shadcn/ui
- Icons: lucide-react
- Forms: react-hook-form + zod validation
- Database/Backend: Supabase or PostgreSQL with Prisma
- Auth: secure authentication with email/password and Google login
- File upload: image upload for listings and office logos
- Dashboard charts: simple analytics cards/charts
- Fully responsive: mobile-first and desktop-ready
- SEO-friendly pages and clean URL structure

LANGUAGE + UX:
- Default language Arabic
- RTL layout everywhere
- Professional Arabic typography
- Elegant cards, shadows, spacing, and premium modern layout
- Fast-loading interface with clean filters
- Make the experience feel like a serious Kuwaiti/GCC proptech startup
